# murkypond

Emergency Forum Failover System for Okoun.cz.

## Modules

- `interceptor/` - userscript bunker UI and failover logic
- `harvester/` - Python/Playwright scraper based on Piker concepts
- `vault/` - Firebase now, self-hosted later
- `schema/` - shared data contracts
- `docs/` - architecture, selectors, compatibility notes
